# 면역 케어 온톨로지 도메인 분석서

## 1. 4계층 면역 케어 아키텍처

### Layer 1: 환경 모니터링 (Environmental Monitoring)

| 환경 요인 | 센서 | 복합지표 | 면역 연결고리 |
|---------|-----|--------|------------|
| PM2.5/PM10 | SPS30 | 산화스트레스 부하 | PM2.5→ROS→NF-κB→IL-6 (r=+0.52, lag 6-24h) |
| VOCs | SGP41 | 산화스트레스 부하 | VOCs→상피장벽 파괴→Th2 편향→IgE (r=+0.35, lag weeks) |
| CO2 | SCD41 | 환기지수 | 환기 품질 간접 지표 |
| 온도/습도 | BME680 | 알레르겐 노출점수 | 고습도→진드기→Der p1→IL-13 (r=+0.30, lag days) |
| BTEX | GC-PID | 산화스트레스 부하 | 면역독성 휘발성 화합물 |
| PFAS | GC-PID | - | 잔류성 면역교란 물질 |
| HCHO | - | 실내공기질 | 실내 자극물질 |
| 라돈 | - | - | 면역억제 |
| 곰팡이/진드기 | - | 알레르겐 노출점수 | 알레르겐 트리거 |
| O3 | - | 산화스트레스 부하 | 산화적 손상 |

**파생 복합지표:**
- 산화스트레스 부하 (Oxidative Stress Load): PM + VOC + O3
- 환기지수 (Ventilation Index): CO2 감쇠율
- 알레르겐 노출점수 (Allergen Exposure Score): 습도 + 곰팡이 + 진드기

### Layer 2: 라이프로그 / 생체신호 (Lifelog / Biosignals)

| 생체신호 | 측정 대상 | 면역마커 상관 | 생리학적 의미 |
|---------|---------|------------|------------|
| HRV | 자율신경 톤 | IL-6과 r=-0.42 | 미주신경 항염증 반사 약화 시 감소 |
| SpO2 | 산소포화도 | CRP와 r=-0.35 | 기도 염증 시 감소 |
| 수면의 질 | Deep/REM 비율 | TNF-α와 r=-0.38 | 수면부족→HPA축→코르티솔→CRP↑ |
| 활동량 | 걸음수, METs | r=-0.28 | 비활동→지방조직→아디포카인→TNF-α↑ |
| 피부온도 | 일주기 리듬 | r=0.31 | 체온 조절, 일주기 무결성 |

**2단계 스마트 면역 모니터링:**
- Stage 1: 웨어러블 연속 모니터링 (24h HRV, 체온)
- Stage 2: SERS 기반 정밀 정량 (CRP, IL-6 분자 수준 측정)

### Layer 3: 바이오마커 (Biomarkers)

**염증성 사이토카인:** IL-6, TNF-α
**항염증 사이토카인:** IL-4, IL-10, IL-13
**알레르기/전신 마커:** IgE, CRP, IL-5, IL-17
**산화 손상 마커:** 8-OHdG, MDA
**신호전달 경로:** NF-κB, Th1/Th2 균형, NLRP3 인플라마좀
**유전자 마커:** 면역 관련 유전자 변이 (2단계 창원대 협업)

### Layer 4: 면역 케어 (Plasma Medicine)

**RONS 종류:**
| 종 | 분자식 | 분류 |
|---|--------|------|
| 오존 | O3 | ROS |
| 하이드록실 라디칼 | •OH | ROS |
| 일중항 산소 | ¹O2 | ROS |
| 일산화질소 | NO | RNS |
| 이산화질소 | NO2 | RNS |
| N2O3 | N2O3 | RNS |
| N2O5 | N2O5 | RNS |

**핵심 인과 체인:** 플라즈마 파라미터 → RONS 생성 → 생체 반응 → 면역 조절

## 2. 질병 궤적 모델 (Disease Trajectory)

**알레르기 마치 경로:**
아토피피부염(L20) → 천식(J45) → 알레르기 비염(J30) → 건선(L40) → 안구건조증(H04.1)

**인과추론 방법:**
- SHAP 분석: 기여도 정량화
- SPARQL 쿼리: 의학적 설명
- GNN: 온톨로지 자동 생성
- XAI: 설명 가능한 AI

**성능 목표:**
- 1단계: 기준선 수립 (AUC 0.75)
- 2단계: AUC ≥ 0.70
- 3단계: AUC ≥ 0.85

## 3. 대상 질환

1. 건선 (Psoriasis) - Th17/TNF-α축
2. 아토피 피부염 (Atopic Dermatitis) - Th2 사이토카인
3. 류마티스 관절염 (Rheumatoid Arthritis)
4. 안구건조증 (Dry Eye Disease)
5. 탈모 (Alopecia)
6. 알레르기 비염 (Allergic Rhinitis) - ICD: J30
7. 천식 (Asthma) - ICD: J45
